# Difference Engine: First Principles

*Difference Engine* is a real-time mixdown generator which outputs a unique render of *Nothing Left to Learn - First Principles* for every page visit. No two sessions are the same.

This idea started with comparing difference live versions of the same songs by the same bands, relative to the original studio recording versions. Live versions can vary wildly, but usually will carry the core of the original version somehow, approximating the original in recognisable ways.

First Principles doesn't have an authoritative original version, and any approximation of its core is as "right" as any other.
